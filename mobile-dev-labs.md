# Методические указания к выполнению лабораторных работ

Предмет: Разработка мобильных приложений

---

## Предварительные требования

Для успешного выполнения лабораторных работ необходимо:
 * Компьютер под управлением ОС: GNU/Linux | Windows10/11 | MacOS
 * Установленное ПО: 
   * Git,
   * VSCode (Можно альтернативные IDE/Текстовые редакторы, официально поддерживаемые разработчиками dart)

## Общие рекомендации

В ходе лабораторных работ вам будет необходимо выполнять шаги из официального ознокомительного руководства по Dart и Flutter.
Чтобы фиксировать прогресс, необходимо создать публичный репозиторий на одной из платформ:
 * [GitHub](https://github.com/)
 * [Codeberg](https://codeberg.org/)
 * Любой другой аналог по предварительному согласованию

Перед выполнением ЛР необходимо создать пустой локальный репозиторий [(инструкция)](https://git-scm.com/book/ru/v2/%D0%9E%D1%81%D0%BD%D0%BE%D0%B2%D1%8B-Git-%D0%A1%D0%BE%D0%B7%D0%B4%D0%B0%D0%BD%D0%B8%D0%B5-Git-%D1%80%D0%B5%D0%BF%D0%BE%D0%B7%D0%B8%D1%82%D0%BE%D1%80%D0%B8%D1%8F) и опубликовать его на выбранной из выше представленного списка платформе.

После выполнения очередного шага руководства при наличии изменений файлов репозитория выполняем команды 
```bash
git add .
git commit -m "Комментарий: небольшой текст, объясняющий суть последних изменений -- зависит от конкретного шага"
git push
```
**!!!комментарий в коммит подставляете свой, а не просто копируете**

Наличие хорошо прокомментированной истории коммитов в публичном репозитории -- **ключевое** требование для успешной сдачи лабораторных работ. 

Если планируете выполнять работы на нескольких компьютерах, ознакомьтесь с [Git Book](https://git-scm.com/book/ru/v2) (достаточно выборочно).

1. В вашем репозитории создайте папку lr1
2. Внутри созданной папки выполните вступительный обучающий курс по Dart: [ссылка](https://dart.dev/learn/tutorial)

## Лабораторная работа №2

1. В вашем репозитории создайте папку lr2
2. Внутри созданной папки выполните вступительный обучающий курс по Flutter: [ссылка](https://docs.flutter.dev/learn/pathway/tutorial)
В рамках лабораторной работы можно (но не обязательно) ограничиться разделом 2: Introduction to Flutter UI
